"Ejercicio 1"

Este programa en Java pide al usuario que ingrese dos números. Luego, compara estos números para encontrar cuál es el mayor y determina si son iguales, si el primero es mayor o si el segundo es mayor. El resultado se muestra al usuario, quien puede decidir si quiere repetir el proceso con otros números o terminar el programa. Utiliza un bucle para permitir múltiples comparaciones y dos métodos auxiliares para realizar las tareas específicas de comparación y clasificación.

"Ejercicio 2"

Este programa en Java simula un juego de adivinanzas en un tablero de 4x4. Al iniciar, se genera un tablero vacío con un barco oculto colocado aleatoriamente. El jugador, sin conocer la ubicación del objetivo, intenta adivinar dónde está ingresando coordenadas de fila y columna. Cada intento se marca en el tablero: un acierto revela el objetivo con un '*', y un fallo se indica con una 'X'. El juego continúa solicitando intentos hasta que el barco es hundido. Finaliza mostrando la ubicación acertada del barco. 

"Ejercicio 3"

Este programa Java genera una tabla de números aleatorios dentro de un rango dado, encuentra el número más alto en la tabla y muestra la tabla completa junto con el número más alto. Utiliza la clase Random para los números aleatorios y recorre la tabla para identificar el mayor valor antes de imprimir los resultados.

"Ejercicio 4"

En el "Juego del Rebaño de Ovejas", te encuentras frente a un tablero virtual de 5x5, simulando un campo donde las ovejas están dispersas. La tarea es simple pero desafiante: reunir todas las ovejas moviéndolas a espacios vacíos adyacentes. Cada movimiento debe ser cuidadosamente pensado, ya que solo puedes mover las ovejas de forma legal y contigua. El objetivo es lograr que todas las ovejas se unan en un solo grupo.

"Ejercicio 5"


Este ejerciocio proporciona funcionalidades para trabajar con un lienzo ASCII. Los métodos permiten dibujar caracteres y rectángulos, cambiar caracteres en el lienzo y mostrar el lienzo completo. El método principal main incluye un bucle que permite al usuario seleccionar diferentes opciones de acción, como dibujar, cambiar caracteres o mostrar el lienzo.
